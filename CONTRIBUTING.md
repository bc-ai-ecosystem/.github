# Contributing to the BC + AI Ecosystem 🛠️

Welcome! We are thrilled you want to build the future of grassroots AI with us. 

Whether you are writing Python scripts for our community bots, formatting data from our Vancouver meetups, or building hardware for a creative art jam, your contributions are what make this ecosystem thrive.

To keep our digital R&D lab organized and accessible, please read through these guidelines before submitting code.

## 🗺️ The Map: Where does your contribution belong?

Because we have a lot of different projects running at once, we use a tiered approach to our GitHub organization:

### 1. You built a cool personal project (The Showcase)
Did you build a local LLM wrapper, a robotic hand, or a generative art tool? We want to see it! 
* **Do not** create a new repository in this organization. 
* **Do** host the code on your personal GitHub account, and submit a Pull Request to our [`awesome-bc-ai`](https://github.com/bc-ai-ecosystem/awesome-bc-ai) repository to get it featured in our community directory.

### 2. You want to help build community infrastructure
Want to help maintain the BC + AI website, improve our Discord bots, or organize our open datasets?
* Check the **Issues** tab on core repositories (like `bc-ai-website` or `discord-whatsapp-bridge`).
* Comment on an issue you want to tackle so we know you're working on it.
* **Fork** the repository, create a branch (`git checkout -b feature/your-feature`), and submit a **Pull Request**.
* A core maintainer will review your code and merge it!

### 3. You are participating in a Hackathon
<!-- tentative planning -->
If you are currently in one of our multi-round hackathons:
* You will be granted temporary write access via a GitHub Team to the specific hackathon repository (e.g., `vanai-hackathon-005`).
* Follow the specific README instructions inside that repository for submitting your team's project.

## 💬 Where We Talk

GitHub is where we store our code, but **Discord is where we collaborate.** If you have an idea for a massive architectural change, a new community tool, or just need help debugging your environment, please start a thread in the [BC + AI Discord](https://discord.gg/vwZzFHpWQ8) *before* spending hours writing code. We want to make sure your time is well spent!

## ⚖️ Code of Conduct

By participating in any BC + AI repository, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).
