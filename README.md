# BC + AI Community Health Files 🌲🤖

This repository contains the default community health files and configuration for the **BC + AI Ecosystem** GitHub organization. 

Because we manage multiple projects—from open datasets to hackathon templates—we use this central repository to store our global guidelines. If a specific repository within our organization does not have its own contributing guide or code of conduct, the files in this repository serve as the default.

## 📂 Repository Structure

* `profile/README.md`: The public-facing landing page for the BC + AI GitHub Organization.
* `CODE_OF_CONDUCT.md`: The overarching rules and ethics we abide by across all projects.
* `CONTRIBUTING.md`: The global guide on how to contribute to our ecosystem, whether you are submitting a personal project to our showcase or fixing a bug in our core infrastructure.

For more information on the BC + AI Ecosystem, please visit our [main organization page](https://github.com/bc-ai-ecosystem) or join our [Discord](https://discord.gg/vwZzFHpWQ8).
